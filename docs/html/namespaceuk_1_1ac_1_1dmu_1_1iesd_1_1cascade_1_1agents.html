Detailed Description

This package contains all the definitions for agents.

It contains sub-packages for prosumers and aggregators. The main classes, which define characteristics that all Prosumers and aggregators must have are abstract classes - ProsumerAgent and AggregatorAgent respecticely

All implementations of Prosumers or Aggregators must extend one of these classes
