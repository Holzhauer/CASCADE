Detailed Description

An AggregatorAgent is an object that represents a commercial/business entity providing energy services to prosumer agents ( ProsumerAgent) such as household prosumers ( HouseholdProsumer) [i.e.

it is involved in retail trade], while at the same time is securing its gross energy need by trading in the wholesale market. The AggregatorAgent class is the abstract superclass of aggregator agents. Examples of aggregator agents include ESCO, RECO and GENCO.

This method should define the pre_step for the agents.

It is assumed that aggregator takes the first step, using this method, usually to send a signal It then processes the behaviour of the prosumers on the sent signal for the same timeslot using this step method This has priority 1 (first), followed by prosumers' step (priority second), and aggregators' step (priority thrid)

This method should define the step for the agents.

It is assumed that aggregator takes the first step, using step_pre() method, usually to send a signal It then processes the behaviour of the prosumers on the sent signal for the same timeslot using this step method This has priority 3, after step_pre and the prosumers' step (priority second)
